# TeleDraw

**TeleDraw is a tiny always-on shared canvas for long-distance connection.**  
Two devices, two people, one drawing — synced in real time over the internet.
---

## What It Is

TeleDraw is a paired drawing display system built with:

- ESP8266 (Wemos D1 Mini)
- 2" SPI TFT displays (ST7789)
- A Node.js WebSocket server
- A web-based drawing interface

You draw on a webpage.  
Your partner sees it appear instantly on their physical screen.
---

## Features

- ✅ **Real-time two-way drawing**
- ✅ **Persistent canvas** (survives reboots & reconnects)
- ✅ **Offline-safe recovery** via periodic full-frame snapshots
- ✅ **Multiple brush sizes & colors**
- ✅ **Live paired devices**
- ✅ **WiFi setup via captive portal**
- ✅ **Automatic reconnect after network drops**
- ✅ **Aesthetic, cozy web UI**
- ✅ **Physical always-on display**

TeleDraw is designed for:
- Long-distance couples  
- Close friends  
- Parents & kids  
- Anyone who wants a quiet shared space
